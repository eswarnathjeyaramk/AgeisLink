#  BLE Crash Detection System

A Flutter-based Crash Detection App integrated with ESP32 using Bluetooth Low Energy (BLE).

## 📌 Overview

This project connects a mobile application to an ESP32 device.
When a crash signal is detected from the hardware, the app:

- Connects via BLE
- Receives real-time alert data
- Fetches live GPS location
- Triggers emergency SOS message

## 🛠️ Tech Stack

- Flutter
- Dart
- ESP32
- flutter_blue_plus
- geolocator
- permission_handler
- url_launcher

## ⚙️ Features

- 🔄 Auto BLE scanning and reconnect
- 📡 Real-time crash signal detection
- 📍 Live GPS location retrieval
- 🚨 SOS message trigger
- 📱 Clean UI with live data display
